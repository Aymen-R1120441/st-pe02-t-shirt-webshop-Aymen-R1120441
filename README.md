# Express Webshop

## Basisvereisten
In de webshop-folder vind je een opdracht terug die gemaakt is geweest in web-design. 

In deze opdracht transformeer je die site naar een server-side gerenderde Express-app met Handlebars (hbs) views, gegenereerd met express-generator. Je past partials toe voor menu, footer en de header op de t-shirts pagina's. Je voegt alle static assets correct toe, en zorgt dat de site responsief blijft werken.
